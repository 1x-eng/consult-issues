# Consult Issue Tracker

[![GitHub issues](https://img.shields.io/github/issues/1x-eng/consult-issues)](https://github.com/1x-eng/consult-issues/issues)

Public issue tracker for [Consult](https://pypi.org/project/getconsult/) — the multi-agent consensus CLI.

## Why This Repo

Consult's source is private. This repo exists so you can:

- Report bugs
- Request features
- Ask questions
- Track known issues

## Before Opening an Issue

1. Check [existing issues](https://github.com/1x-eng/consult-issues/issues) — someone might've hit the same thing
2. Include your consult version (`consult --version`)
3. Include relevant error output (redact API keys)

## What Consult Is

Multi-agent consensus system. N LLM agents analyze your problem in parallel, critique each other, iterate until agreement, then synthesize one answer.

```bash
pip install getconsult
```

Docs and usage: [getconsult.sysapp.dev](https://getconsult.sysapp.dev)
